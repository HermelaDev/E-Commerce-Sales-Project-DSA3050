## 📝 Project Overview
This repository contains an end-to-end E-commerce Sales Analysis dashboard. The project demonstrates a full Business Intelligence lifecycle, from sourcing real-world data to deploying a live, cloud-based solution.

### 🛠️ Key Components
* **Data Transformation**: Utilized Power Query to clean and prepare a dataset of over **7,000 rows**. Transformations included handling null values, correcting data types, and creating derived time-intelligence columns.
* **Data Modelling**: Designed and implemented a **Star Schema** with a central Fact table and descriptive Dimension tables. Relationships were configured with **Many-to-one (*:1)** cardinality and **Single** filter direction for optimal performance.
* **Dashboarding**: Developed a professional, interactive executive summary. Features include **KPI summary visuals**, **Monthly Revenue Trends**, and **Category Distribution** analysis.
* **Interactivity**: Integrated **Date** and **Regional slicers** with cross-filtering functionality to support data-driven decision-making.
